# CICS Automation Track — Batch 2026

**Official repository for the CICS Automation Track Batch 2026**
*IoT Enablement Project for the Common Room at Frassati Building, 20th Floor*

---

## Introduction

The **CICS Automation Track Batch 2026** is an academic initiative under the College of Information and Computing Sciences aimed at designing and implementing IoT-driven automation solutions. This project serves as the capstone effort of Batch 2026, bringing together embedded systems, cloud connectivity, and intelligent automation under one cohesive development pipeline.

## Project Context

This repository supports the **technological enablement and IoT infrastructure development** of the Common Room established at the **Frassati Building, 20th Floor**. The initiative transforms the space into a smart, connected environment through sensor integration, actuator control, and real-time data monitoring — making the Common Room a living laboratory for automation technologies.

## Objectives

- Design and deploy a functional IoT automation system for the Common Room.
- Establish reliable communication between embedded devices, backend services, and user-facing interfaces.
- Create a scalable and maintainable codebase that future batches can extend and improve.
- Document all hardware configurations, software architecture, and deployment procedures.
- Foster collaborative development through version control best practices.
