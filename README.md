## Introduction

The **CICS Automation Track Batch 2026** is an academic initiative under the College of Information and Computing Sciences aimed at designing and implementing IoT-driven automation solutions. This project serves as the capstone effort of Batch '26, bringing together embedded systems, cloud connectivity, and intelligent automation under one cohesive development.

## Project Context

This repository supports the **technological enablement and IoT infrastructure development** of the Common Room established at the **Frassati Building, 20th Floor**. The initiative transforms the space into a smart, connected environment through sensor integration, actuator control, and real-time data monitoring, making the Common Room a living laboratory for automation technologies.
